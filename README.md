# Chat Application (MERN Stack)

A real-time chat application similar to WhatsApp built using the MERN stack.

##  Features
- Real-time messaging
- User authentication
- Responsive design
- REST API integration

##  Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

##  Project Structure
- web → Frontend (React)
- server → Backend (API & logic)
